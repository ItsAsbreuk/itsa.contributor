How to start becoming a project-member

Being a project-member means you are owner an quardkeeper of ITSA's modules. Project-members are those who judge and merge pull requests. You can only become a project-member if you are asked to and if all current project-members agree. All project-members have equal rights.

The project-members follow the following rules:

  • Merging pull requests will only be done when at least 2 project-members give positive advise and there is no negative advise
  • New modules need to conform the vision and quality of ITSA's modules
  • New project-members can be proposed if all members agree
