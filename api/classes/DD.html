_setupMouseEv

() private

Defined in src/drag-drop/drag-drop.js:964

Available since 0.0.1

Engine behinf the dragdrop-cycle. Sets up a mousedown listener to initiate a drag-drop eventcycle. The eventcycle start whenever one of these events happens on a HtmlElement with the attribute dd-draggable="true". The drag-drop eventcycle consists of the events: dd-start, emitterName:dd-drag and emitterName:dd-drop

dd-fake-mouseup

private

Defined in src/drag-drop/drag-drop.js:144

Available since 0.1

Fired when the mouse comes back into the browser-window while dd-drag was busy yet no buttons are pressed. This is a correction to the fact that the mouseup-event wasn't noticed because the mouse was outside the browser.

valuechange

Defined in src/drag-drop/drag-drop.js:244

Available since 0.1

Fired when the checkbox changes its value
Listen for this event instead of 'checkedChange', because this event is also fired when the checkbox changes its 'disabled'-state (switching value null/boolean)

Event Payload:

  • e EventFacade

    Event Facade including:

    • newVal Boolean | Null

      New value of the checkbox; will be 'null' when is disabled.

    • prevVal Boolean | Null

      Previous value of the checkbox; will be 'null' when was disabled.
